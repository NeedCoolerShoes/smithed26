# NCRS Smithed Summit 26

Data and resource packs for Smithed Summit 2026.

Requires the Smithed Summit 2026 Booth standard library.

Minecraft 26.2

## Load Booth
Make sure Worldedit is installed. Add the schematic to world, located at `assets/ncrs_booth.schem`.

1. `/execute in minecraft:overworld run tp @s -30.53 69.00 -209.60 -180.66 90.00`
2. `//perf off`
3. `schem load ncrs_booth.schem`
4. `//paste`
